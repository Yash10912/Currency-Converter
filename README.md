# Currency-Converter

A simple and interactive currency converter built using HTML, CSS, and JavaScript. This project fetches real-time exchange rates using the Fetch API and demonstrates the use of modern JavaScript concepts such as async/await and Promises for handling asynchronous operations.​


Features
Convert between multiple currencies using up-to-date exchange rates.
Clean and responsive user interface.
Uses async/await and Promises for API calls, making the code more readable and maintainable.​
Easy to use: Enter an amount, select source and target currencies, and get instant results


The API used here is taken from https://github.com/fawazahmed0/currency-api.
